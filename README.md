# Application Overview

---

This is a simple web application that informs users cryptocurrency data. A user can search an cryptocurrency and can see the data regarding it. 

# How to run the app

---

1. Download ZIP
2. cd '/coinsearch'
3. npm install
4. npm start


# Built With

---

- Technology: HTML, CSS, JS, React.js
- Framework: Bootstrap
- API: Cryptocompare
